# AI-Based Crop Disease Detection using CNN

## Overview
A CNN-based deep learning system that classifies plant leaves as healthy or diseased.

## Technologies Used
- Python
- TensorFlow/Keras
- OpenCV
- NumPy
- Streamlit

## Features
- Image preprocessing
- CNN-based classification
- Real-time prediction using Streamlit

## Accuracy
~97%

## Dataset
PlantVillage Dataset

## Model File
The trained model file (`crop_model.h5`) is not included in this repository due to GitHub file size limitations. The model can be regenerated by running the training notebook.

## How to Run

pip install -r requirements.txt

streamlit run app.py
